Проект для дипломной работы по теме " Разработка системы для ручной и автоматической
 сортировки объектов с помощью компьютерного зрения на
 примере прототипа промышленного робота-манипулятора". В которой было разработано Android приложение, запрграмированн микроконтроллер Arduino для управлениия роботом-манипулятором, смоделирован и распечатан на 3D-принтере робот-манипулятор и рабочее окружение к нему.
